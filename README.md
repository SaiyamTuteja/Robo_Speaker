# Text-to-Speech Project

This is a simple Python project that uses the `pyttsx3` library for text-to-speech conversion.

## Usage

1. Clone the repository:

    ```bash
    git clone https://github.com/SaiyamTuteja/Robo_Speaker.git
    ```

2. Navigate to the project directory:

    ```bash
    cd your-repository
    ```

3. Install the required library:

    ```bash
    pip install pyttsx3
    ```

4. Run the script:

    ```bash
    python robospeaker.py
    ```

5. Enter the text you want to hear. To exit, type 'q' and press Enter.

## Dependencies

- Python 3.x
- pyttsx3 library
